⌨️ Typing Speed Test Game

A fun and interactive Typing Speed Test Web App built using HTML, CSS, and JavaScript.
This project helps users measure their typing speed (WPM) and accuracy in real-time.

🌐 Live Demo

👉 https://typing-speeds-game.netlify.app/

🚀 Features

⌨️ Real-time typing test

⏱️ Timer-based test

📊 Words Per Minute (WPM) calculation

🎯 Accuracy tracking

🔄 Restart / Try Again option

🎨 Clean and responsive UI


🛠️ Tech Stack

HTML5
CSS3
JavaScript (Vanilla JS)

⚙️ How It Works

User starts typing the given text
Timer begins automatically
App tracks:
Typing speed (WPM)
Accuracy
Final result is shown after completion
